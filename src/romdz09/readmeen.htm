The manual is easy to start

1. Set java, if no
2. run the program - click on it
3. button 'new' find your ROM.
4. button 'extract' he unpacks ROM.
5. The next 'get pictures' - A new window will appear. There appeared fingering parts of the ROM, see the different resources.
----- All buttons have hints , go to the button and its description will appear .....


Extract button saves the image in the 'files' folder (in the root directory where is stored ROM) with the name is the combined selected file name
and it place (hex offset). It can be edited and put with the same name. When you import used this file .

Next subtlety. It replaces the original location in the file with the extension .b, where it is compressed by the 'Chinese original algorithm'.
Therefore, it must be less than the original (but already there). But this is not the whole complexity. The fact is that with the extension .b file then again compressed LZMA algorithm
and than it is written in the ROM.
Pictures remain separate color table. Therefore, the less colors get the smaller file. Well, the more smooth and repetitive blocks will be better.
Incidentally repetition reduce the size even in different pictures, if they are in the same file with extension .b.

On the firmware NX9_61D_240_V2_1_ZSX_DZ09_32_T3_L3_C4_G4_B_160512 after exporting the font is issued in the form of one long picture. (Mono-color, black and background)
Each letter is placed in the red frame inside it you can edit it as you like (I made a bold). Further imports.
